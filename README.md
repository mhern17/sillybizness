# Pro Stock Analyzer Browser App With Definitions

This version adds definitions next to/under the values and hover tooltips.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud

Upload these files to GitHub:

- app.py
- requirements.txt
- README.md
- .gitignore

Then set the Streamlit Cloud main file path to:

```text
app.py
```
